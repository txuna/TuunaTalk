# TuunaTalk
 
### frontend 
1. html, css, vanilla javascript, socket.io 

### backend 
1. node.js - express framework, socket.io 

### Logic 
socket.io를 이용한 간단한 채팅앱
express-session을 이용하여 세션을 획득한 사용자만 chat app(socket.io)을 이용할 수 있도록 제작 

### Usage 
node index.js
